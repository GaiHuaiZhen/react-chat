## Terminal

```sh
$ git clone git@github.com:liukaixin01/react-chat.git
```
```sh
$ cd react-chat
```
```sh
$ npm install
```

## 运行 MongoDB

```sh
$ cd /usr/local/mongodb/bin
```
```sh
$ sudo ./mongod
```

## 运行 node

```sh
$ npm run node
```

## 启动项目

```sh
$ npm start
```
