## Install

> `git clone git@github.com:liukaixin01/react-chat.git`

> `npm install`

## mongodb

- #### Mac OSX 平台安装 MongoDB
- sudo brew install mongodb
- sudo mkdir -p /data/db
- sudo mongod
- 连接不上请查看：/usr/local/var/log/mongodb/mongo.log

## Dev

> `npm start`

> `nodemon server/server.js`
